下面哪个不是合法的变量？

- [ ] `$f11`
- [ ] `$_11f`
- [x] `$11f`

超全局变量的可访问性是怎样的？

- [ ] 仅在函数内可访问
- [ ] 除了类中，其余地方都可访问
- [ ] 因为被定义为`GLOBAL`，所以任意地方都可访问
- [x] 以上都不对

> 自行查看*超全局变量*一节。
