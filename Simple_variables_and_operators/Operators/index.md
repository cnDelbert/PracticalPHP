# 操作符

操作符对操作数进行操作，改变传入的值。例如，在`2 + 3`中，2和3是操作数，`+`是操作符。有三种操作符：一元、二元和三元操作符。一元操作符只有一个操作数，二元操作符有两个操作数，三元的有三个。如你所见，`+`是二元操作符，将两个变量作为输入。

PHP中有大量操作符，你需要重点记忆14个。

首先是最重要的操作符。不用一次全部记忆，后面有例子来帮你记忆：

|操作符|说明|
|:-:|:-:|
|+|二元，将操作数一同操作数二相加。|
|-|二元，从操作数一种减去操作数二。|
|*|二元，将操作数一同操作数而相乘。|
|/|二元，操作数一除以操作数二。|
|.|二元，将字符串一同字符串二相连。|
